This is my first blog post. Written in Markdown, built with Hugo, deployed on Vercel.

## Why Blog?

Writing helps me think. Publishing forces clarity.

## Stack

- **Hugo** — fast static site generator
- **PaperMod** — clean, minimal theme
- **Vercel** — auto-deploy on git push

## Workflow

```bash
# Write a new post
hugo new content posts/my-new-post.md

# Preview locally
hugo server -D

# Publish
git add . && git commit -m "new post" && git push
```

That's it. No database, no CMS, no nonsense.
